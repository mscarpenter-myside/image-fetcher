Instalação Passo a Passo

Para começar a usar o MySide Image Fetcher, siga as etapas abaixo. O processo leva menos de 2 minutos.

1

Baixar o Projeto

Acesse o Google Drive para baixar o arquivo ZIP da extensão, ou clone o repositório GitHub.

Baixar Extensão
2

Descompactar

Extraia o conteúdo do arquivo .zip em uma pasta de sua preferência no computador (ex: Documentos/Extensoes/MySide).

3

Abrir o Chrome

No navegador Google Chrome, copie e cole o endereço abaixo na barra de navegação:

chrome://extensions/
4

Modo do Desenvolvedor

No canto superior direito da página de extensões, ative a chave "Modo do desenvolvedor".

5

Carregar Extensão

Clique no botão "Carregar sem compactação" no canto superior esquerdo. Selecione a pasta onde você descompactou os arquivos — a pasta que contém o arquivo manifest.json.

6

Fixar a Extensão

Clique no ícone de Extensões (peça de quebra-cabeça) ao lado da barra de endereços do Chrome e clique no pin ao lado do MySide Image Fetcher para fixá-lo permanentemente.

2

Como Utilizar

Capturando Imagens

Navegue em qualquer site e, ao encontrar uma imagem que deseja baixar:

1

Clique com o botão direito sobre a imagem.

2

Selecione a opção "Enviar para MySide Image Fetcher".

3

O painel lateral abrirá automaticamente com a imagem carregada.

Configurando a Nomenclatura SEO

No painel lateral, você encontrará os campos para processamento:

1

Tipo de Imagem

Selecione entre Capa (máx 100kb) ou Corpo (máx 150kb). A compressão ideal é aplicada automaticamente.

2

Descrição (Opcional)

Dois campos para adicionar detalhes descritivos da imagem, como tipo de vista ou nome do empreendimento.

3

Bairro e Cidade

  • Autocomplete: Sugestões aparecem conforme você digita. Confirme com Tab ou Enter.
  • Auto-Preenchimento: Cidade preenchida automaticamente para bairros únicos na base.
  • City Picker: Seletor visual para bairros presentes em múltiplas cidades, com navegação por teclado.

Processar e Baixar

Clique no botão "Processar e Baixar". A extensão irá:

1

Redimensionar para exatamente 747px de largura

2

Comprimir para o limite selecionado com qualidade máxima

3

Renomear como descricao-bairro-cidade.jpg

4

Abrir a janela de salvamento do sistema operacional

3

Dicas

Temas Personalizados

No topo do painel, alterne entre os temas Dark, Light e o MySide theme.

Correção Rápida

Errou a cidade no seletor? Clique novamente no campo Cidade para reabrir as opções disponíveis para aquele bairro — sem precisar redigitar nada.

Atalhos de Teclado

Use Tab para aceitar sugestões, ↑ ↓ para navegar no City Picker, e Enter para confirmar.
